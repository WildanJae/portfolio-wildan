# Portfolio — Wildan Jaelani

Personal portfolio website. Clean, minimal, responsive.

## Stack

- Next.js 14 — App Router
- TypeScript
- Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

## Kustomisasi

Edit `lib/data.ts` untuk mengubah profil, skill, pengalaman, dan project.

Foto profil ada di `public/wildan.jpg` — ganti file-nya untuk update foto.
